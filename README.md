##About
Brainfuck-C is a [Brainfuck](http://en.wikipedia.org/wiki/Brainfuck) interpreter written in C.

##Example
```
gcc brainfuck.c -o bf
echo "++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>." | ./bf
```
Also try running [this](http://www.99-bottles-of-beer.net/language-brainfuck-1718.html) code.
